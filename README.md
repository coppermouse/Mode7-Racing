# Fake Mode7 demo for pygame
* warning: performance bad :(

## requirements: 
* pygame 1.9.6 (pygame.org)
* pytmx 3.21.7 (https://pypi.org/project/PyTMX/)

## how to play:
* Execute "game.py" to start the game
* Control the kart with WASD
